I also work with Docker and Docker Compose to create containerized development environments and manage multi-service applications. This approach allows me to build reproducible, scalable, and portable development setups for web services and backend infrastructures. Alongside backend development, I explore workflow automation tools such as n8n, where I design automated pipelines that connect APIs and streamline processes between different services and applications.

In addition to backend systems, I have experience building mobile applications using Kotlin and Firebase, enabling me to develop Android applications with cloud connectivity and real-time data synchronization. My development workflow relies heavily on Git and GitHub for version control, collaboration, and maintaining well-structured repositories for my projects and portfolio work.

Overall, I enjoy building practical, scalable, and efficient software systems while continuously expanding my knowledge in DevOps, cloud technologies, backend architecture, and automation engineering. I am passionate about learning new tools, solving complex technical problems, and contributing to projects that improve efficiency through technology.
